📋 How to Add a New KB Entry

All KB entries are stored in data/kb-data.json in your GitHub repository. To add a new entry:

Steps
  1. Go to GitHub → data/kb-data.json
  2. Click ✏️ Edit button
  3. Copy the template below
  4. Paste inside the "entries" array
  5. Fill in your data
  6. Update "version" field
  7. Commit → Vercel auto-deploys ✅

  "num": "KB38",
  "title": "Your Document Title",

this document covers.",
