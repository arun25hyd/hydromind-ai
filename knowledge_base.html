How to Use Knowledge Base
Click to expand — 4-step guide + search tips
▼
1
Search by Keyword, Model, or KB Number
Type any keyword into the search bar — component model (e.g. A4VG90), manufacturer name (e.g. Rexroth), document type (e.g. service manual), or directly enter a KB number (e.g. KB-003). Results update instantly as you type.
2
Filter by Category
Use the filter chips to narrow results by document type — Cranes, Pumps, Motors, Valves, Standards, or Maintenance. Each chip shows a count of available documents. Combine filtering with search for precise results.
3
Sort and Switch Views
Sort the results by KB Number, Title A-Z, Manufacturer, searchable chunks, or page count using the sort dropdown. Toggle between Grid view (cards) and List view (compact rows) using the view buttons on the right of the result bar.
4
Query a Document with AI Advisor
Click "Ask AI" on any KB card to open the AI Advisor pre-loaded with that document's context. The AI will answer your specific fault or technical question using that OEM manual as its primary reference — no need to read the full document.
💡 Search Tips
Search by component model number for the most precise results — e.g. "A4VG90", "H1B", "D3FW", "K3VL".
Use KB-XXX format to jump directly to a specific document if you already know the number from an AI Advisor response.
Search "counterbalance" or "CBV" to find all valve setting guides across manufacturers in one filtered view.
The Standards filter shows ISO, DNV, API, and engineering rule references — useful for pre-job class requirement checks.
Click Load More at the bottom to reveal all indexed documents beyond the initial 12 shown.
