How to Use AI Hydraulic Advisor
Click to expand — 5-step guide + pro tips
▼
1
Select Your Mode
Choose from the four tabs at the top — Hydraulic Advisor (fault diagnosis), KB Search (OEM manual lookup), Calculator (hydraulic sizing), or System Design (guided 12-step design). Each mode changes the AI prompt and input placeholder automatically.
2
Filter by Manufacturer (Optional)
Use the brand chips in the left panel to narrow the AI context to a specific OEM — Rexroth, Danfoss, Parker, Eaton, Liebherr, MacGregor, NOV, SeaTrax, or Favelle Favco. Selecting a brand focuses the KB search on that manufacturer's documents.
3
Describe Your Fault or Query Precisely
Include: system type (crane, winch, steering), affected circuit (hoist, luff, slew), component model if known, actual pressure readings (bar), and whether the fault is intermittent or constant. The more specific your input, the more actionable the diagnosis.
4
Read the Structured AI Response
Responses follow a structured format — Root Cause → Test Points → Corrective Action → KB References. KB tags appear below each AI message and are clickable — tap one to immediately ask a follow-up about that document.
5
Use Suggested Questions to Get Started
Tap any of the suggestion cards on the welcome screen for one-click fault starters. Or use the Quick Hint chips below the input box for common fault prompts.
💡 Pro Tips
Always quote actual pressure readings (bar/PSI) from your gauges — the AI uses these numbers for precise fault isolation.
Use Calculator mode for step-by-step hydraulic sizing with formulas shown.
Ask the AI to "Run a full 10-state diagnostic sequence" for structured fault isolation.
Shift+Enter for a new line — Enter alone sends the message.
For KB Search: "Rexroth A4VG90 swashplate control" returns more targeted results than a generic description.
